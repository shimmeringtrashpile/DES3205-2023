This text is inside an html tag with a <p> tag.

We have a 'p' selector in our styles.css file the the property value pair of 'color: blueviolet;' so our text is blueviolet.

Our 'p' selector is also setting our text-alignment to left 'text-alignment: left;' so our text is left-justified in the browswer.

Also, our font-size has been increased to 18 pixels (px). The browser's default font-size is 16px. If we didn't define a font-size property, we would be seeing this text at 16px.
